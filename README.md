# Disaster Recovery (Next.js Project)

This project is built with **Next.js**, **Tailwind CSS**, and other modern tools.  
Follow the instructions below to run the project locally.

---

## Prerequisites

Make sure you have the following installed on your system:

- **Node.js** (v22.15.0 or later)
- **npm** (10.9.2 or later)
- **pnpm** (v10.17.0 or later)

You can verify the versions by running:

```bash
node -v
npm -v
```

Installation
Download zip folder, extract files and open it in vscode:
```bash
cd disaster-recovery
```
Install dependencies using pnpm:
```bash
pnpm install
```
Development Server
Start the development server:
```bash
pnpm dev
```
The app will run on 
```bash
http://localhost:3000
```
